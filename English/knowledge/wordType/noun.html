# 根据其意义

  1. 专有名词 ( 主要指人名、地名及某类人和事物专有的名词 )
    • 人名 -> Mr. Green | Hemingway
    • 地名 -> China | Beijing | the Yellow River
    • 某类人的名称 -> Americans | Russians
    • 某类事物的名称 -> Buddhism（佛教） | English
    • 月份、星期、节日等 -> January | Monday | New Year's Day
    • 书名、电影、诗歌等 -> Gone with the Wind（随风而逝） | Venom（毒液）
  2. 普通名词 （指一类人、事物、物质或抽象概念的名称，这些名词一般不用来指某一具体事物）
    1. 个体名词
      1. 指作为个体而存在的人或东西
        • 可以指具体的人或物:
          • He has two sisters. 他有两个姐姐。
          • Pandas live in the forest. 熊猫生活在森林里。
        • 也可以指抽象的人或物:
          • A new century has just begun. 一个新的世纪刚刚开始。
          • I had a dream last night. 我昨天晚上做了一个梦。
      2. 个体名词有复数的形式
        • weeks | problems
      3. 个体名词单数形式可以和a/an连用
        • a week | a problem
    2. 集体名词
      指一群人或一些事物总称，表示由个体组成的集体
      • family | team | class | audience
    3. 物质名词
      指无法分为个体的东西
      • tea | cloth | rain
    4. 抽象名词
      表示一些抽象概念，用来指人或事物的品质、情感、状态等
      • honesty | love | silence

# 根据语法特征

  1. 可数名词
    1. 定义：
      • 可用数目计算的名词
      • 个体名词、集体名词大多是可数名词
    2. 复数的规则
      1. 一般在词尾加-s
        book -> books | pen -> pens
      2. 以s、x、sh、ch结尾的词，在词尾加-es
        bus -> buses | box -> boxes | brush -> brushes | watch -> watches
      3. 以辅音字母+y结尾的词，变y为i，再加-es
        baby -> babies | city -> cities
      4. 以f或fe结尾的词，变f或fe为v，再加-es
        knife -> knives | wife -> wives
      5. 以o结尾的词，一般加s，有的加-es
        Negro -> Negroes| hero -> heroes | tomato -> tomatoes | potato -> potatoes | mosquito -> mosquitoes | echo -> echoes | volcano -> volcanoes
    3. 不规则变化
      1. 变 -a- 为 -e-
        man -> men | woman - women
      2. 变 -oo- 为 -ee-
        foot -> feet | tooth -> teeth | goose -> geese
      3. 变 -ouse 为 -ice
        mouse -> mice | louse -> lice
      4. 词尾加 -(r)en
        ox -> oxen | child -> children
  2. 不可数名词
    1. 定义
      • 不可用数目计算的名词。
      • 物质名词、抽象名词大多为不可数名词。
    2. 用法
      • 前面不可以加 a、an
      • 前面不可以加 one、 two、 three..
      • 后面不可以加 s 、永远是单数
    3. 英语中常见的不可数名词有
      furniture家具，equipment设备
      bread面包，knowledge知识，machinery机器，advice建议
      progress进步，baggage行李，luck运气，wealth财富

# 名词所有格

# 有生命的物体的名词所有格

  1. 一般在单数名词词尾直接加's
    • 如：Tom's mother
  2. 不以s或es结尾的复数名词，在词尾加's'。
    • 如：Children's Day
  3. 以s或es结尾的复数名词，只在词尾加'。
    • 如：the students' classroom学生们的教室

# 无生命的物体的名词所有格

  1. 无生命的物体的名词所有格通常由of + 名词构成。
    • 如：the name of our school我们学校的名字
  2. 表示时间、距离、重量、国家、城市等无生命的名词，也可以加's'或'。
    • 如：five minutes' walk 5分钟的路程
    • 如：a stone's throw away一箭之遥

# 名词所有格口诀

英语名词所有格，表示某物是谁的。
所有格构成有方法，单数名词后加 's。
复数名词词尾有s/es，只在词尾加 '。
名词若是无生命，所有格构成用 of。
