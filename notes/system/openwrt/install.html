# 步骤

  1. 安装pe工具到u盘

  2. 操作（文件夹/文件名 尽量简单，后续需要命令行输入）

    1. 创建一个文件夹 （aa）

    2. 解压工具包并将下面四个文件复制到 aa

    3. 解压 openwrtOS ,将 ...eft.img 移动到 aa

    4. 推出U盘，然后插入待刷硬件设备，开机进入bios，设置U盘为启动项，没有的话就禁止ssd/hd启动

  3. 进入pe系统

    1. 打开命令行窗口 (win+r)。
    2. 切换到aa目录 (一般在C盘 )
      • 切换C盘 cd C:
      • 查看当前文件夹 dir
    3. 输入命令：physdiskwrite.exe -u ...eft.img，其中 ...eft.img 是系统镜像名称。
    4. 选择要安装的磁盘编号，一般为0。
    5. 输入Y开始安装。
    6. 按装完成后就拔出U盘，重新启动硬件，进入bios设置启动项，正常来说可以进入openwrtOS
