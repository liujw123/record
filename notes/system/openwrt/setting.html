# openwrt - setting

# 设置静态IP地址

设置后重启路由，按照新ip进行后台登陆

  1. 可以在 linux 界面输入 vim /etc/config/network

  2. 用网线连接 路由器 & 电脑（多网口情况下，最左边|最右边，多尝试），然后输入默认ip 进行设置

# 设置 wan & lan 口

​ 全部操作后记得点击保存&应用，最好重启一下

  1. 先排序 （0为wan口，其它全部勾选为lan口，注意桥接模式勾选）

    ​

  2. 配置 pppoe 拨号

# 连接光猫

  • 如果光猫是桥接直接网线连接wan口后，lan口链接设备就可以上网了
  • 如果不是自行找方法改成桥接

# 配置WIFI

  1. 连接wifi路由器，登陆后台
  2. 上网方式改成AP模式，DHCP服务器关闭
  3. 配置wifi账号密码就可以了
